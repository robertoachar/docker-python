# docker-python

[![License][license-badge]][license-url]

> A playground for Docker with Python.

# Development

* Cloning the repo

```bash
$ git clone https://github.com/robertoachar/docker-python.git
```

# Docker

* Building an image

```bash
$ docker-compose build
```

* Running a container

```bash
$ docker-compose up
```

* Stopping a container

```bash
$ docker-compose down
```
